# hotspots-ranked
investment property hotspots ranked

## install

```
npm install
```

## configure

copy `.env.sample` as `.env` and edit settings

## logs

create folders:

```
logs/lettings
logs/sales
```

## run using ts-node

```
npm run start:dev
```

## build

```
npm run build
```

## run using node

```
npm run start
```

## test

```
npm run test
npm run test:coverage
```
